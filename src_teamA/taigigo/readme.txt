対義語（のようなもの）を生成するプログラム。
※あまり精度は高くない。。。
下記URLを参考にして作成したもの。（対義語ファイルは自作）
https://qiita.com/youwht/items/f21325ff62603e8664e6
辞書データは自分で作成するか、別途Webから持ってくる必要がある。

詳細：
Word2Vecを利用しWikipediaに記載の全言語をベクトルで表現。
A：「対象の言葉」、B：Aの類似語、C:Bの対義語　とした場合、
A - C + B で抽出される単語を対義語として抽出している。
※対義語を上記式で表現する定義があるわけではなく、
  いくつか試してみた結果、この式が当てはまったというだけ。
